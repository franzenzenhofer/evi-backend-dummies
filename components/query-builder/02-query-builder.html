02. Query Builder - Suchkonfiguration

Definieren Sie die "Hintergrundssuche" für diese Landing Page. Alle Besucher sehen Ergebnisse basierend auf diesen Filtern.

Datenquelle

Such-Operatoren: - = NOT (ausschließen), + = MUST HAVE (erforderlich), OR = ODER-Verknüpfung
Sucht nur in Titeln. Leer lassen für alle Titel.

Wenn aktiviert, werden die geografischen Filter unten ignoriert.
